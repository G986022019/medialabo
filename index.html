本テーマの学習目的

学習目標:

  • HTML によって Web ページの構造を記述する基本的な方法がわかる
  • CSS によって Web ページの見た目を設定する基本的な方法がわかる
  • JavaScript によって Web ページのインタラクティブな処理を記述する基本的な方法がわかる
  • HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できる．
  • HTML, CSS, JavaScript, 画像・音声・動画ファイルを組み合わせて簡単な Web ページが作成できる

プロジェクト課題ではこの授業を学んだ知識を全面的に考察する課題と思います。毎回の授業でこの課題を少し書いたが、まだ大変時間かかりました。 私作ったページは天気を検索するページで、最も苦労したところはCSSだと思います。bugがないことはwebページの基本です。その上で、良いページと 言われるため、CSSが大切なことと言えます。JavaScriptとJavaが似ているので、そんな難しくないと思います。しかし、JSの書き方がいっぱいある ので、最も簡単な書き方が重要です。その簡単さが出来なかったと思います。最後はHTMLですが、毎回の授業で80%ができたと思うので、後にはdiv要素を増えることだけです。

数当てゲーム

この課題ではプレイヤーが数値をテキストボックスに入力して、ボタンを押して、入力した数値と答えの数値が同じかどうかを判定する課題です。 3回目の授業で、デバッグ用のWebページを作りました。まずは入力回数を表示します。その次、二つの数値が同じかどうかを判定するJSを作りました。 そして、入力回数が3より大きいの場合、答えを教えてあげることを設定しました。そして、4回目の授業で、実際にWebページで入力した予想と判定の文を 表示しました。3回目の授業で書いたJSを少し修正したら、出来上がりました。意外に簡単だと思います。1回分の予想しかないが、5回目で完全品を作れると思います。 5回目の授業でいよいよこのWebページを完成しました。今回は10桁以下のコードしか書きませんでしたが、文字から関数を変換するコードが分かりませんでした。 自分で長い時間を考えて、やってみったが、まだできませんでした。Web上で調べたら、意外に難しくないです。
