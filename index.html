本テーマの学習目的

学習目標:

  • HTML によって Web ページの構造を記述する基本的な方法がわかる
  • CSS によって Web ページの見た目を設定する基本的な方法がわかる
  • JavaScript によって Web ページのインタラクティブな処理を記述する基本的な方法がわかる
  • HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できる．
  • HTML, CSS, JavaScript, 画像・音声・動画ファイルを組み合わせて簡単な Web ページが作成できる

数当てゲーム

この課題ではプレイヤーが数値をテキストボックスに入力して、ボタンを押して、入力した数値と答えの数値が同じかどうかを判定する課題です。 3回目の授業で、デバッグ用のWebページを作りました。まずは入力回数を表示します。その次、二つの数値が同じかどうかを判定するJSを作りました。 そして、入力回数が3より大きいの場合、答えを教えてあげることを設定しました。そして、4回目の授業で、実際にWebページで入力した予想と判定の文を 表示しました。3回目の授業で書いたJSを少し修正したら、出来上がりました。意外に簡単だと思います。1回分の予想しかないが、5回目で完全品を作れると思います。 (200字以上)
